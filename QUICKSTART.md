# Quick Start Guide

## Get Your Website Running in 3 Steps!

### Step 1: Install Dependencies
```bash
cd marketing-agency
npm install
```

### Step 2: Start Development Server
```bash
npm run dev
```

### Step 3: Open in Browser
Navigate to: http://localhost:3000

## Next Steps

1. **Customize Content**: 
   - Update company name in `components/Navigation.tsx`
   - Edit services in `components/Services.tsx`
   - Modify stats in `components/Hero.tsx`

2. **Change Colors**:
   - Find and replace the gradient colors throughout components
   - Current: `#667eea` and `#764ba2`

3. **Add Your Logo**:
   - Replace "CreativeHub" text in `components/Navigation.tsx`
   - Or add an image logo

4. **Deploy**:
   - Push to GitHub
   - Deploy on Vercel (free hosting for Next.js)
   - Or use any hosting platform
